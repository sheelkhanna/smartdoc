SmartDoc

SmartDoc is an outline based document creation and publishing tool. It is designed to make structured, understandable documents easily and quickly. SmartDoc can be useful for story writing, FAQ creation, novel planning, speech-writing, manual writing, software support, biographies, and lesson planning.
The basic idea in SmartDoc is that you can include many separate notes within a single file. This means that you do not need to create several files - for most purposes it is enough to create only one file and hold all your notes inside it. With the addition of the tree-type notes, you now have a three-dimensional notebook: many notes within one file and a multi-level, nested pages within a single note.

SmartDoc Editor and Viewer

SmartDoc Web Editor is full WYSIWYG editor to create and publish SmartDoc document. It is easy to share and use SmartDoc, you can embed it on any webpage.

SmartDoc XML Structure

SmartDoc documents are stored as XML file. It is very easy to compose and publish SmartDoc file. 

<smartdoc name="Example 1" id="Example_1226280853468" title="Welcome to SmartDoc">
	<smarttab title="First Tab" icon="icons/docs/file/2.gif">

		<node title="Default Node">
			<data><![CDATA[<P>The formatted text in default node.</P>]]></data>

		</node>

		<node title="Second Node" icon="icons/docs/file/1.gif">
			<data><![CDATA[<P>Second node text.</P>]]></data>

			<node title="Child Node">
				<data><![CDATA[<P>Child node text.</P>]]></data>

			</node>

		</node>
	</smarttab>
	<smarttab title="Second Tab" icon="icons/docs/file/11.gif">

		<node title="Default Node" icon="icons/docs/multimedia/7.gif">
			<data><![CDATA[Add text here..]]></data>

		</node>

	</smarttab>
</smartdoc>
